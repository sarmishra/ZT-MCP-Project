# ZT-MCP: Zero-Trust Security Architecture for MCP-Connected AI Agents

**Reference Implementation for IEEE COINS 2026**

This repository contains the reference implementation of **ZT-MCP**, a zero-trust enforcement middleware designed specifically for the Model Context Protocol (MCP) ecosystem. It provides mathematically auditable, protocol-level access control to prevent tool spoofing, unauthorized invocation, prompt injection, and data exfiltration.
